PHP

Niveau :todo

Ma définition

Hypertext Preprocessor, plus connu sous son sigle PHP, est un langage de programmation libre, principalement utilisé pour produire des pages Web dynamiques via un serveur HTTP, mais peut fonctionner comme n'importe quel langage interprété de façon locale.

Selon les données de W3Techs, PHP est utilisé par 78,9% de tous les sites web avec un langage de programmation côté serveur connu. Ainsi, près de 8 sites Web sur 10 visités sur Internet utilisent PHP d’une manière ou d’une autre. Ce qui fait de PHP un langage qui intéresse encore beaucoup d’entreprises.

Malgré 30 ans d’existence il reste dans le TOP 10 des langages les plus populaires d’après le classement du TIOBE.

Mes éléments de preuve

En ce qui concerne mon expérience avec PHP, j’ai eu l’occasion de l’utiliser dans plusieurs projets.

Etant donné les services web que propose l’entreprise de mon alternance j’ai beaucoup utilisé PHP, notamment pour le projet de MACS GDPR avec une API Symfony. Cette api PHP sert à interagir avec une ou des bases de données pour authentifier des utilisateurs, la récupération, l’insertion et la mise à jour des données.

Un autre projet pour SOLIHA Nouvelle-Aquitaine m’a permis d’approfondir mes connaissances en PHP, l’objectif était de réaliser des opérations CRUD (CREATE, READ, UPDATE, DELETE) sur leur base de données MICROSOFT SQL SERVER en fonction des appels fait par leur logiciel de comptabilité SAGE.

Mon autocritique

Grâce à mon expérience sur ce langage, j’estime avoir un niveau confirmé et je peux répondre aux besoins de n’importe quelle entreprise éditrice de logiciel WEB.

J’ai l’autonomie pour faire des modifications et évolutions sur du code existant.

PHP est une compétence essentielle pour un profil de développeur web, sachant que la plupart des sites web utilisent PHP, je suis donc satisfait d’avoir un niveau confirmé avec ce langage.

Ayant acquis beaucoup d’expérience avec ce langage, je ne peux que vous conseiller de l’utiliser dès que possible pour la réalisation d’un projet personnel ou professionnel. Cependant, si vous êtes débutant dans la programmation commencez d’abord par apprendre les bases de l’affichage d’une page web avec le langage HTML, puis ensuite tournez-vous vers PHP et apprenez comment faire des conditions, des boucles, des tableaux et ensuite comment communiquer avec une base de données.

PHP est un langage en constante évolution, toujours maintenu et développé malgré 30 ans d’existence il est encore largement utilisé sur Internet. J’ai donc conscience que l’on peut toujours en apprendre davantage, c’est pourquoi je continue à me tenir informé des évolutions du langage en maintenant une veille technologique. Je compte acquérir le livre de Olivier Heurtel sur la nouvelle version PHP 8, je compte aussi dès que possible suivre une formation de ENI Service sur le développement avancé avec PHP.
