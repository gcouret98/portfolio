Javascript

Niveau : Confirmé

Ma définition

JavaScript est un langage de programmation utilisé principalement comme langage de script des pages web. Souvent abrégé en « JS », à ne pas confondre avec le langage Java, les deux sont des marques déposées par Oracle mais ont une syntaxe et des usages différents. Si vous souhaitez implémenter des fonctionnalités complexes sur une page web, vous devrez utiliser JavaScript. Il permet d’ajouter de l'interactivité à votre site web. Sa notoriété augmente grâce à l’utilisation de ses frameworks comme Angular, Vue JS ou React.

J’ai pu expérimenter Angular sur un projet puis Vue JS sur deux autres projets. Ces frameworks sont idéals pour proposer un affichage web dynamique avec des données provenant d’une API par exemple.
JavaScript est toujours d’actualité et le restera sûrement.

Mes éléments de preuve

JS est le langage avec lequel j’ai commencé la programmation, notamment avec le jeu « Keep Clean », il s’agit de mon premier projet informatique, c’est un jeu mandaté par une association à l’occasion d’une fête sur les jeux rétros. L’objectif du jeu est de trier les déchets qui tombent du haut de l’écran avec la bonne poubelle jaune ou verte. La difficulté principale de ce projet était la gestion des collisions avec la corbeille et les déchets. Ce projet était une bonne manière de se lancer dans le bain pour comprendre les possibilités qu’offre ce langage.

Ensuite dans un contexte professionnel j’ai pu utiliser Vue JS pour le projet MACS GDPR. J’ai vite assimilé la syntaxe et les bonnes pratiques grâce à un projet déjà existant. Son utilisation est logique si elle est destinée à un site web complexe avec des données et un utilisateur réalisant des actions.

Par exemple pour la saisie d’un formulaire, vous devez dessiner l’emprise des travaux sur une carte et mon code JavaScript permet dans cet exemple d’envoyer un objet géométrique (le dessin) à une API qui s’occupe de trouver les parcelles les plus proches. J’ai apprécié travailler sur une carte interactive, je trouve la fonctionnalité très utile, elle facilite la saisie de l’utilisateur et permet aussi de mieux normaliser la donnée saisie.

Mon autocritique

Si vous avez comme projet d’être développeur web vous allez utiliser JavaScript, ça fait maintenant deux ans que je pratique ce langage je pense avoir un niveau confirmé, il est difficile d’être un expert sur ce langage qui est très vaste. Vue JS m’a beaucoup plu je vous conseille ce documentaire si vous êtes intéressé par l’histoire de ce Framework. JavaScript occupe une place importante pour un profil de développeur web et grâce à mes expériences je suis maintenant à l’aise avec ce langage. Je me tiens informé des nouveautés, il est important de rester au courant des changements d’usages avec les nouvelles versions du Framework que vous utilisez, il se peut que la syntaxe change drastiquement par exemple avec Vue 3 et Vue 4. Étant donné mon utilisation hebdomadaire de ce langage je n’envisage pas une formation supplémentaire. Selon la difficulté rencontrée j’effectue les recherches nécessaires pour résoudre le problème. En plus, j’ai eu la chance durant mon alternance d’avoir des collaborateurs avec beaucoup d’expérience qui m’ont transmis leurs connaissances.
